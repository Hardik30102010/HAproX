# HAproX IDE
***
![logo](Logo.png)
***
A Compiler specially for HTML , CSS and JAVASCRIPT.
## WHAT FUNCTIONS CAN IT DO ?
* HAproX IDE has a plain text writer from where you can download your code/text file, copy your code/text, save it with extension, and change its name.
* It has a JavaScript compiler ( JAVASCRIPT CONSOLE) where you can write javascript and, it will run automatically.
* It has JavaScript Canvas where you can write code for the canvas tag
